## ❓ FAQ

### General Questions

**Q: Is RLSearch safe to use?**
A: Yes! RLSearch is completely safe and does not modify game files. It only reads game memory for statistics. No VAC ban risk.

**Q: Does RLSearch work with Steam and Epic Games?**
A: Yes, RLSearch supports both Steam and Epic Games versions of Rocket League.

**Q: Can I use RLSearch in casual matches?**
A: Yes, but it's optimized for competitive (ranked) matches.

**Q: Will RLSearch get me banned?**
A: No. RLSearch does not inject into the game or modify any files. It's external monitoring only.

### Activation & Premium

**Q: How do I activate premium?**
A: Enter your username and activation code in the format XXXX-XXXX-XXXX on the login screen.

**Q: Can I use one code on multiple computers?**
A: No. Each code is locked to one computer via HWID. This prevents code sharing.

**Q: What happens when my premium expires?**
A: You'll receive a notification and need to activate a new code to continue using RLSearch.

**Q: Can I transfer my code to a new computer?**
A: No. HWID locking prevents code transfer. Contact support for special cases (hardware replacement).

**Q: Do you offer refunds?**
A: Yes, within 24 hours of purchase if the code hasn't been activated. Contact sales@rlsearch.com.

### Technical Questions

**Q: Why does RLSearch need administrator rights?**
A: Administrator rights are required to read game process memory and create system-level hooks for monitoring.

**Q: What is HWID?**
A: Hardware ID - a unique identifier generated from your computer's hardware (CPU, motherboard, disk serial).

**Q: Can I run RLSearch on multiple monitors?**
A: Yes! The overlay window can be moved to any monitor.

**Q: Does RLSearch affect game performance?**
A: No. RLSearch uses minimal resources (< 100 MB RAM, < 1% CPU).

**Q: What ports does RLSearch use?**
A: RLSearch only needs internet during activation. No ports needed during normal use.

### Troubleshooting

**Q: Game not detected?**
A: 
- Ensure Rocket League is running
- Run RLSearch as Administrator
- Check Windows Firewall isn't blocking
- Restart both RLSearch and game

**Q: Overlay window not appearing?**
A:
- Check if Rocket League is in Windowed/Borderless mode
- Verify administrator rights
- Try closing and reopening overlay

**Q: Statistics not updating?**
A:
- Click "STOP" then "START" again
- Close and reopen overlay window
- Restart RLSearch

**Q: "Code already used" error?**
A:
- Each code works on only ONE computer
- Check if you used it on another PC
- Contact support if you believe this is an error
